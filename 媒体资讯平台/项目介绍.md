技术栈：Java 21、Spring Boot 3.x、Spring Framework 6.x、MyBatis 3.x、PageHelper 2.x、Redis 7.x、MySQL 8.4 LTS / 9.x 驱动、JJWT 0.12.x、Spring Task、AOP、阿里云 OSS、WebSocket

项目介绍：面向媒体行业的一站式新闻资讯系统，涵盖稿件审核、内容分发、数据看板等核心模块，包含spring boot后端与web管理端与用户端（前端都采用Vue 3）。

**核心职责：**

认证权限设计：基于 JWT 实现多端登录鉴权，通过 ThreadLocal 封装用户上下文工具类，保证线程安全与用户信息隔离。

Redis缓存：针对资讯热点排行、分类列表等高频接口设计 Redis 缓存策略，大幅降低数据库 IO 压力，接口响应速度提升显著。

AOP 代码解耦：自定义切面实现审计日志记录、公共字段（创建人 / 更新人 / 时间）自动填充，减少冗余代码

定时任务调度：基于 Spring Task 实现资讯热度过期清理、凌晨自动统计前日阅读数据，支撑数据看板报表生成。

数据与存储优化：使用 MyBatis + PageHelper 实现高效分页查询；接入阿里云 OSS 托管图片等静态资源，降低服务器带宽消耗。

版本约束：后端生成代码必须基于 Java 21 与 Spring Boot 3.x，统一使用 `jakarta.*` 命名空间，不再使用 Spring Boot 2.x / Java 8 / `javax.*` / `io.jsonwebtoken:jjwt:0.9.x` 等旧版写法。

实时消息推送：集成 WebSocket 实现稿件审核状态后台实时推送；使用 Apifox 完成接口文档维护与自动化测试。
